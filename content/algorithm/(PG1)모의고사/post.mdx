## 문제

[코딩테스트 연습 - 모의고사](https://programmers.co.kr/learn/courses/30/lessons/42840)

<br />

## 코드

```jsx
function solution(answers) {
  const answer = [];
  const score = [0, 0, 0];
  const student1 = [1, 2, 3, 4, 5];
  const student2 = [2, 1, 2, 3, 2, 4, 2, 5];
  const student3 = [3, 3, 1, 1, 2, 2, 4, 4, 5, 5];

  for (let i = 0; i < answers.length; i++) {
    answers[i] === student1[i % student1.length] && score[0]++;
    answers[i] === student2[i % student2.length] && score[1]++;
    answers[i] === student3[i % student3.length] && score[2]++;
  }

  for (let i = 0; i < score.length; i++) {
    if (score[i] === Math.max(...score)) answer.push(i + 1);
  }

  return answer;
}
```

<br />

## 문제 풀이 방법

학생별로 같은 패턴을 반복하기 때문에 `나머지(%) 연산자`를 사용하여 배열을 split 하면서 비교한다.
