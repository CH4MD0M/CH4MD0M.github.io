## 문제

[코딩테스트 연습 - 크레인 인형뽑기 게임](https://school.programmers.co.kr/learn/courses/30/lessons/64061)

## 코드

```jsx
function solution(board, moves) {
  let answer = 0;
  const stack = [];

  for (let i = 0; i < moves.length; i++) {
    for (let j = 0; j < board.length; j++) {
      if (board[j][moves[i] - 1] !== 0) {
        if (board[j][moves[i] - 1] === stack[stack.length - 1]) {
          stack.pop();
          answer += 2;
        } else {
          stack.push(board[j][moves[i] - 1]);
        }
        board[j][moves[i] - 1] = 0;
        break;
      }
    }
  }
  return answer;
}
```

## 문제 풀이 방법

`stack` 을 사용하여 문제를 해결했다.

`board` 에서 `stack` 으로 값을 `push` 하고 같은 것이라면 `stack` 에서 `pop` 한다.

그리고 선택한 board의 값은 반복문이 끝나기 전 0으로 처리한다.
