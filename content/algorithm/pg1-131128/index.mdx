## 문제

[코딩테스트 연습 - 숫자짝꿍](https://school.programmers.co.kr/learn/courses/30/lessons/131128)

## 코드

```jsx
function solution(X, Y) {
  const hashX = [...Array(10)].fill(0);
  const hashY = [...Array(10)].fill(0);

  [...X].forEach((item) => hashX[item]++);
  [...Y].forEach((item) => hashY[item]++);

  const answer = hashX.map((item, idx) => {
    if (item !== 0 && hashY[idx] !== 0) {
      return String(idx).repeat(Math.min(item, hashY[idx]));
    }
  });

  const result = answer.reverse().join("");

  return result ? (+result ? result : "0") : "-1";
}
```

## 문제 풀이 방법

`hash Table`을 이용하여 구현하였다.<br/>
X, Y에 대해 각각 hash Table을 만들고 X, Y의 요소들로 hash Table을 업데이트한다.<br/>
`hashX`와 `hashY`의 공통적으로 있는 요소를 찾는다.(교집합)

문제 조건에 맞게 0으로만 이루어져 있으면 0을, 짝이 존재하지 않는 다면 -1을, 짝이 존재한다면 결과값을 return 한다.

> 쉬운 문제였지만 이제는 단순히 반복문을 사용하려 하지 말고 자료구조를 알맞게 선택하도록 노력해야겠다.
