## 문제

[코딩테스트 연습 - 압축](https://school.programmers.co.kr/learn/courses/30/lessons/17684)

<br />

## 코드

```jsx
function solution(msg) {
  const answer = [];
  // 사전 초기화
  const dict = new Map(
    [...Array(26)].map((_, i) => {
      return [String.fromCharCode(i + 65), i + 1];
    })
  );

  let w = (c = 0);

  while (true) {
    c += 1;

    if (c === msg.length) {
      answer.push(dict.get(msg.slice(w, c)));
      break;
    }

    if (!dict.has(msg.slice(w, c + 1))) {
      dict.set(msg.slice(w, c + 1), dict.size + 1);
      answer.push(dict.get(msg.slice(w, c)));
      w = c;
    }
  }

  return answer;
}
```

<br />

## 문제 풀이 방법

- 사전(dict)을 Map으로 정의하여 초기화하였다.
- w와 c변수를 선언하여 투포인터로 문제를 해결하였다.

<br />

`msg` 에서 순서대로 글자를 불러오는데,<br/>
**현재 글자 + 다음 글자** 가 사전에 등록되어 있지 않다면 사전에 등록하고 `answer` 배열에 값을 `push` 한다. <br/>
**현재 글자 + 다음 글자** 가 사전에 등록되어 있다면 `c` 를 증가시킨다.

<br />

`c`가 `msg`의 길이와 같을 때(즉, 마지막일 때)는 `answer` 배열에 값을 `push`하고 `while` 문을 빠져나온다.

<br />

## 새로 알게된 것

- `Map()` 객체의 길이를 구하기 위해서는 `Map.prototype.size` 접근자 프로퍼티를 사용한다.

```jsx
const map = new Map();
map.set("a", 1);
map.set("b", 2);
map.set("c", 3);

console.log(map.size); // 3
```
