## 문제

[코딩테스트 연습 - 신규 아이디 추천](https://school.programmers.co.kr/learn/courses/30/lessons/72410)

<br />

## 코드

```jsx
function solution(new_id) {
  let answer = new_id
    .toLowerCase()
    .replace(/[^\w\d-_.]/g, "")
    .replace(/\.{2,}/g, ".")
    .replace(/^\.|\.$/g, "");

  if (!answer.length) answer = "a";
  answer = answer.slice(0, 15).replace(/\.$/g, "");

  const len = answer.length;

  return len > 2 ? answer : answer + answer[len - 1].repeat(3 - len);
}
```

<br />

## 다른 사람 코드

```jsx
function solution(new_id) {
  const answer = new_id
    .toLowerCase()
    .replace(/[^\w\d-_.]/g, "")
    .replace(/\.{2,}/g, ".")
    .replace(/^\.|\.$/g, "")
    .padEnd(1, "a")
    .slice(0, 15)
    .replace(/^\.|\.$/g, "");

  return answer.padEnd(3, answer[answer.length - 1]);
}
```
